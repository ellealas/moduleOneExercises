Git – local configuration

You may remember we asked you to create a GitHub account, this will come in handy soon.
Git is extremely big, and important. Remember, your work and your business’ work may be here,
so you want to make sure that it is always protected. Run the following commands to configure
your account details.
Open the command prompt (the Terminal in VS Code will do).


Run these commands to configure your username and email:
git config --global user.name "Your Username"
git config --global user.email "youremail@email.com"
